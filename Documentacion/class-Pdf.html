Behaviour: dest, indicates where send the documment. It can bo one of following 'I': send the file inline to the browser. The PDF viewer is used if available. 'D': send to the browser and force a file download with the name given by name. 'F': save to a local file with the name given by name (may include a path). 'S': return the document as a string. name, The name of the file. It is ignored in case of destination S. The default value is doc.pdf. $this->format, Indicates if name is encoded in ISO-8859-1 (false) or UTF-8 (true). Only used for destinations I and D. The default value is false.
